# Ver-0.021B — Client Home Lock and Popup Style Note

## Booking prefill fix
Signed-in client sessions now re-apply the client profile into Step 1 booking details when the session is restored, and again when the booking flow is opened.

## Locked client-facing home page
The current client app home page is considered locked:
- premium top bar with ZipBook icon and account/login icon
- simple welcome screen
- Make a booking button
- login/sign-up opens in a popup

Do not alter the client-facing home page without warning Brian first, unless he explicitly requests the change.

## Popup styling standard for future popups
Future popups should follow the Ver-0.021 popup standard:
- centred vertically and horizontally
- sticky title/header
- sticky bottom action area where applicable
- neat scrollbar contained inside the popup body
- bleed space below sticky header
- comfortable bottom bleed space
- premium spacing on desktop and mobile
