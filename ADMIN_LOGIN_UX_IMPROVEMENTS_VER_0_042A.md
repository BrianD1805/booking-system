# ZipBook Ver-0.042A — Admin Login UX Improvements Follow-up

Ver-0.042A applies the requested admin login polish after Ver-0.042.

## Changes

- Stopped the password reveal eye button from moving on mouseover, focus or click.
- Replaced emoji reveal buttons with proper inline eye icons.
- Hidden passwords now show the reveal icon as an eye with a line through it.
- Visible passwords show the normal eye icon.
- Reduced the desktop Master key heading size to match the Staff login heading treatment.
- Removed the local testing fallback helper text from under the master key input.

## Database

No SQL changes required.
