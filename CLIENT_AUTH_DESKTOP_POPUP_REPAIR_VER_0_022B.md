# ZipBook Ver-0.022B — Desktop Login/Sign-up Popup Repair

Ver-0.022B keeps the locked client-facing home page unchanged and applies a desktop-only repair to the client login/sign-up popup.

## Changes

- Widened the desktop login/sign-up popup so the form has more room.
- Reduced desktop-only header sizing, spacing, tab sizing, form spacing and footer button height.
- Moved the signed-in/signed-out status out of the lower popup area and into a compact status pill beside the login/sign-up header on desktop.
- Kept the mobile breakpoint styling unchanged so the Ver-0.021C mobile popup behaviour is not disturbed.
- Left the separate forgot-password popup foundation from Ver-0.022A intact.
- Bumped visible version to Ver-0.022B and service worker cache to zipbook-v0.022B.

## Database

No new database migration is required.
