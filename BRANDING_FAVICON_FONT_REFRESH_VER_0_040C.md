# ZipBook Ver-0.040C — Favicon Revert and Ubuntu Font

Ver-0.040C restores the favicon/app icon artwork to the attached original blue-background ZipBook logo and changes the app font family throughout to Ubuntu.

- Replaced favicon, Apple touch icon, PWA icons, maskable icons, Open Graph image, and shared icon master using the attached original logo.
- Updated global app font stack to Ubuntu.

- Kept the existing Ver-0.040 colour/theme refresh unchanged.
- Bumped visible app version to `Ver-0.040C`.
- Bumped service worker cache to `zipbook-v0.040C`.
- Bumped package version to `0.0.40-c.0`.

No SQL migration is required for Ver-0.040C.
