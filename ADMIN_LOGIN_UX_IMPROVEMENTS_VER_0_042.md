# ZipBook Ver-0.042 — Admin Login UX Improvements

Ver-0.042 improves the admin login experience.

## Changes

- Split the master key and staff login into separate screens on desktop and mobile.
- Staff login appears only after the master key has been accepted.
- First staff creation remains on its own screen after master key unlock when no staff users exist.
- Reduced the Staff login heading size on desktop.
- Increased spacing between staff login fields and the sign in button.
- Moved the desktop staff sign in button to the right to align visually with the master key unlock action.
- Added password reveal eye controls for the master key and staff password fields.
- Staff password reveal defaults to visible, as requested.

## SQL

No SQL required.
